# Curso de Frontend e React do Básico ao Avançado

Este repositório apresenta os projetos desenvolvidos durante o curso de **Frontend e React** oferecido pela **Udemy**. Este curso é ideal para quem deseja aprender **desenvolvimento frontend** desde o início até técnicas mais avançadas, utilizando tecnologias modernas como **React.js**.

## Visão Geral

O curso é estruturado de forma progressiva, começando com projetos mais simples e avançando para aplicações mais complexas. Essa abordagem permite que os alunos construam uma base sólida e evoluam gradativamente, adquirindo as habilidades necessárias para criar interfaces de usuário robustas e dinâmicas.

## Destaques do Curso

- **Projetos Iniciais:**
  - Foco em HTML, CSS e JavaScript.
  - Introdução aos conceitos básicos de desenvolvimento web.
  - Criação de layouts responsivos e estilizados.

- **Projetos Intermediários:**
  - Introdução ao **React.js**.
  - Componentização e reutilização de código.
  - Utilização de **props** e **state**.

- **Projetos Avançados:**
  - Integração com APIs externas.
  - Gerenciamento de estado com **Redux**.
  - Criação de aplicações completas com navegação dinâmica.

## Projeto em Destaque: Maneva

O primeiro projeto do curso apresenta uma página simples sobre a banda **Maneva**, que utiliza HTML e CSS para destacar:

- **Cabeçalho:** Informações sobre a banda e uma imagem destacada.
- **Integrantes:** Lista dos membros da banda.
- **Biografia:** História e conquistas da banda.
- **Curiosidades:** Informações adicionais e citações relevantes.
- **Links:** Acesso ao site oficial da banda.

> O projeto enfatiza a construção de uma base sólida com **HTML semântico** e **CSS modularizado**, preparando o terreno para projetos mais avançados.

## Tecnologias Utilizadas

- **HTML5:** Para estruturação das páginas.
- **CSS3:** Para estilização e responsividade.
- **JavaScript:** Para interatividade.
- **React.js:** Para criação de interfaces dinâmicas e reativas (em projetos avançados).

## Benefícios do Curso

- Aprendizado progressivo e estruturado.
- Projetos práticos que simulam situações reais de mercado.
- Desenvolvimento de habilidades essenciais para atuar como desenvolvedor frontend.
